# Mental Health Sentiment Analyzer

## Overview

Mental Health Sentiment Analyzer is an NLP and Large Language Model (LLM) powered analytics pipeline that examines Reddit discussions related to student life, academic pressure, and mental health.

The system processes textual posts, identifies emotional tendencies, determines potential root causes, assesses severity levels, and generates analytical reports that help uncover broader mental health patterns within online communities.

Rather than performing simple positive/negative sentiment analysis, the project focuses on understanding underlying psychological indicators and contributing factors behind mental health concerns.

---

## Features

### Emotional Tendency Detection

Identifies one or more psychological tendencies within a post, including:

* Anxiety
* Depression
* Frustration
* Burnout
* Stress
* Hopelessness
* Fear of Failure
* Low Self-Esteem
* Loneliness
* Academic Pressure
* Positive
* Neutral

### Root Cause Analysis

Determines potential contributing factors such as:

* Exam Pressure
* Family Expectations
* Financial Stress
* Placement Rejections
* Career Uncertainty
* Skill Gap
* Social Isolation
* Relationship Issues
* Health Issues
* Toxic Environment
* Lack of Guidance

### Severity Assessment

Each post is categorized into one of four severity levels:

* Low
* Medium
* High
* Critical

### Confidence Scoring

Provides a confidence estimate for every classification generated by the language model.

### Automated Reporting

Generates:

* Classified post dataset
* Root cause statistics
* Mental health tendency statistics
* PDF summary report

## Installation

### Clone Repository

```bash
git clone <repository-url>
cd Mental_Health_Sentiment_Analyzer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Start Ollama

Ensure Ollama is installed and running locally.

Pull the required model:

```bash
ollama pull qwen3:8b
```

---

## Configuration

Environment variables can be used to customize execution.

| Variable     | Default                             |
| ------------ | ----------------------------------- |
| CSV_FILE     | sample_data/sample_reddit_posts.csv |
| OLLAMA_MODEL | qwen3:8b                            |
| OLLAMA_HOST  | http://localhost:11434              |

Example:

```bash
export OLLAMA_MODEL=qwen3:8b
export CSV_FILE=data/reddit_posts.csv
```

---

## Running the Project

```bash
python mental_health_analyzer.py
```

Generated files will be stored inside the `outputs` directory.
